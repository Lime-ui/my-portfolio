# Murungi Precious Portfolio

A modern personal portfolio website showcasing my skills, projects, experience, and professional background in IT Support, UI/UX Design, and Digital Solutions.


## Technologies Used

* HTML5
* Tailwind CSS
* JavaScript
* Alpine.js

## Features

* Responsive modern design
* Project showcase section
* Skills and experience highlights
* Contact section
* Smooth and clean user interface

## About Me

I am an IT Support Specialist and UI/UX Designer with experience in technical support, systems configuration, troubleshooting, and digital product design. I enjoy building clean, functional, and user-focused digital experiences.

## Installation

Clone the repository:

```bash
git clone https://github.com/Lime-ui/my-portfolio.git
```

Open the project folder and launch the `index.html` file in your browser.
